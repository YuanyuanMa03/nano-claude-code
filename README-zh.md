## 项目简介

`nano-claude-code` 是一个轻量级、实用的 AI 编程助手项目，从零开始构建。

### 核心特性

- 实用工具导向
- 渐进式开发
- 多语言实现（Python、TypeScript、Rust）
- 桌面 GUI + Web 应用
- 多模型支持
- 通用语言支持
