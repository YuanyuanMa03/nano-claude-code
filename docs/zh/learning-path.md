# 学习路径指南

本指南帮助你从学习会话过渡到实际实现。

## 推荐学习路径

### 路径 1: 快速上手 (适合有经验的开发者)

```
1. 阅读 docs/plans/ 中的设计文档
   ↓
2. 运行 sessions/v01_session.py 体验核心循环
   ↓
3. 查看 implementations/python/v01_minimal_loop/ 的完整实现
   ↓
4. 运行测试验证理解
   ↓
5. 继续学习 v02, v03...
```

### 路径 2: 渐进式学习 (适合初学者)

```
1. 从 sessions/v01_session.py 开始
   - 阅读代码注释
   - 运行并尝试不同任务
   - 修改代码观察变化
   ↓
2. 学习 v01_minimal_loop/ 的实现
   - 对比会话和实现的区别
   - 理解模块化设计
   - 运行测试
   ↓
3. 重复上述步骤学习 v02, v03...
   ↓
4. 尝试自己实现小功能
   ↓
5. 阅读设计文档理解整体架构
```

## 会话 vs 版本对照

| 会话 (Sessions) | 版本实现 (Implementations) | 用途 |
|----------------|-------------------------|------|
| `v01_session.py` | `v01_minimal_loop/` | 学习核心循环 |
| `v02_session.py` | `v02_tool_system/` | 学习工具系统 |
| `v03_session.py` | `v03_task_planning/` | 学习任务管理 |
| `v04_session.py` | `v04_code_understanding/` | 学习代码分析 |

## 学习目标检查清单

### v01: 最小循环
- [ ] 理解 `stop_reason` 的作用
- [ ] 能够解释为什么需要循环
- [ ] 知道如何添加新工具
- [ ] 理解消息累积的重要性

### v02: 工具系统
- [ ] 理解工具调度的价值
- [ ] 知道如何实现路径安全
- [ ] 能够添加新的文件操作工具
- [ ] 理解参数验证的重要性

### v03: 任务规划
- [ ] 理解为什么需要 Todo 系统
- [ ] 知道如何实现约束规则
- [ ] 能够实现提醒机制
- [ ] 理解状态持久化的需求

### v04: 代码理解
- [ ] 理解项目结构分析
- [ ] 知道如何实现代码搜索
- [ ] 能够构建依赖关系图
- [ ] 理解静态分析基础

## 实践项目

学习完会话后,尝试这些实践项目:

### 初级
1. **文件管理器**
   - 使用 v01 创建、读取、编辑文件
   - 使用 v02 的文件工具改进

2. **代码重构助手**
   - 使用 v03 规划重构任务
   - 使用 v04 分析代码结构

### 中级
1. **测试生成器**
   - 使用 v04 分析函数
   - 使用 v02 生成测试文件
   - 使用 v03 跟踪进度

2. **文档生成器**
   - 使用 v04 扫描项目
   - 使用 v02 生成文档
   - 使用 v03 管理任务

### 高级
1. **完整 CI/CD 助手**
   - 结合所有版本的功能
   - 实现自动化工作流
   - 添加错误处理和恢复

2. **多语言项目助手**
   - 使用 v07 的语言检测
   - 适配不同语言的工作流
   - 统一的项目管理
